# 8x8ledDisplay
Arduino library for displaying text in 8x8 LED Matrix (Without MAX7219 Driver).

This library has 3 styles to display text on 8x8 LED Matrix.
(Blink, Typewriter and Scroll)
Speed of text can also be altered.
